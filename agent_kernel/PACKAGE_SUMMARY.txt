================================================================================
                            快速开始
================================================================================

1. 自动设置（推荐）
   Linux/Mac: ./setup.sh
   Windows:   setup.bat

2. 手动设置
   python3 -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   cp .env.example .env
   # 编辑.env，填入API密钥

3. 运行
   python agent_kernel.py

4. 运行示例
   python examples/example_basic.py
   python examples/example_team.py
   python examples/example_tasks.py
